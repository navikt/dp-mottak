# dp-mottak

Ansvar for å håndtere hendelser på tema Dagpenger via Joark og sørge for videre saksgang


## Komme i gang

Gradle brukes som byggverktøy og er bundlet inn.

```
./gradlew build
```

## Dokumentasjon

Tilstander se [tilstander](docs/arkitektur/tilstander)
